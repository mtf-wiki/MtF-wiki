## 术后即时考虑

术后以流食、半流食逐渐适应饮食，出院后应食用高蛋白食物，保证丰富的营养，以利于伤口愈合、身体复原。一些医院术后禁食的要求是由于担心排泄、排遗污染术区。（一般禁食七天，期间予以抗生素治疗和营养液补充养分）

## 活动

术后避免运动一般以避免缝合处开裂和避免感染等为目的。

术后一段时间内,术者需要卧床和回避运动。术者卧床的时间应随外科医生(出于防范预后风险的必要)要求、术式和护理条件而定。卧床时，应适当翻身，以避免褥疮。平卧时，避免使用腰枕，以防止血流聚集于会阴缝合口处造成不必要的问题.[^来源未知 群友说的]

不再需要长时间卧床之后,不要做剧烈运动,避免容易导致生殖器部位急性过度拉伸的运动，包括瑜伽、有氧舞蹈、普拉提、游泳、下蹲和举重物，及任何可能对生殖器部位造成压力的活动，如，爬山上坡、骑马、骑自行车或骑摩托车，及避免水中运动。

但应适当短距离散步以促进恢复。

### 就坐

术后一个月内，就坐可能感到不适，但并非不安全。建议采用环形圈垫 （即“甜甜圈”）缓解手术部位压力。环形垫圈应在购物网站上或医疗器械商店以"尾椎骨折垫"或"痔疮垫"等关键词购买。若无医生许可，应避免使用U型的睡眠颈枕，以免不小心坐偏了对术处造成压力。

### 出行

尽管一些术者的身体恢复状况足以在出院时就足以长时间行走并自行乘坐交通工具，考虑到避免运动的要求和突发情况，我们仍然建议您在不得不乘坐交通工具时使用轮椅,尤其是当您在您不熟悉的地区时。

若您乘坐飞机,建议您和您的医院联系(一些医院/诊所会主动提供一份可递交给航空公司的轮椅申请书,一些医院/诊所提供将您转运到机场的服务,一些医院/诊所则不会),提前预定机票,并查询对应航空公司提供的服务，提前1-3日内向出发机场/航空公司预约轮椅等disable aid surveice，在出行当天较平时早若干小时到达机场，由机场或航空公司工作人员将您转运上飞机。头等舱商务舱机票并非必要选择。

若您乘坐火车,国内在12306app上可申请**重点旅客**。

另请注意，请勿在短期内预料到需要乘坐火车飞机等实名制交通工具出行的情况下匆忙于当地修改证件，会造成不必要的麻烦。

## 关注方面:卫生工作与阴道分泌物

术后,尤应重视卫生工作.一方面,良好的卫生情况能有效地预防术后感染,另一方面,恢复期身体无异味无不适感受有助于改善术者的心理健康状况.

### 洗浴/淋浴

术后住院(或于指定酒店居住)期间，医院会提供不同级别的护理服务，可能包括擦洗身体等。

在医生允许，可以酌情进行淋浴(shower,不将下身进入水中)，可参考 与痔疮手术/包皮环切术 类似的护理要求，即:外阴部分伤口愈合之后可以碰水。卫生要求高者可以考虑携带/购买 与痔疮手术/包皮环切术 类似的塑料防水袋。

同时,一般要求在扩张前后特别清洗手术部位,亦不应在洗澡时佩戴模具。

对于沐浴(bash，俗称泡澡)尤其是长时间浸入水中或使用入浴剂的沐浴，需要谨慎。绝大部分医院持保守看法，不建议在术后短期内泡澡; 部分医院对术后不可沐浴(泡澡)时间的建议从一个月到三个月不等。尤其是，即使外观上看起来已经愈合, 当手术部位仍然可以观察到可吸收缝线时，不要在浴缸里泡很长时间。长期暴露在温水中会软化皮肤，使任何残留的缝线更容易撕开。

 沐浴湿巾、身体湿巾、 **干洗洗发水**（如果您经常洗头，当您无法淋浴时，您可能会发现这很有用。）

淋浴或洗浴的流程:

1. 像平常一样洗澡
2. 用少量抑菌物质溶液(包括聚维酮碘洗剂、洁尔阴洗剂、氯己定洗剂等) 清洗会阴,(不必清洁阴道)，然后用清水将抑菌液洗掉.注意聚维酮碘会灼伤皮肤,不宜过度使用；其余洗剂亦会破坏阴道菌群、减慢愈合速度，不宜每天应用。
3. 用婴儿湿巾或毛巾而不是卫生纸通过轻拍而非擦拭的方式吸干会阴水分.
4. 在整个会阴(从阴蒂到阴唇)涂抹抑菌霜

对于一般的身体洗浴(淋浴)  或卧床期间迅速清洁身体，可以准备的方便物品有:

| 卫生用品类别 | 洗浴用品                                                     |
| ------------ | ------------------------------------------------------------ |
| 免洗用品     | 干洗洗发水和婴儿湿巾，沐浴湿巾、身体湿巾(长期卧床时使用)，**抗菌皂、洗手液** |
| 淋浴用品     | 淋浴软管和头(以保证手术部位干燥)  长柄沐浴海绵(以避免弯腰)，下体沐浴护套， |

另请勿在术后短期内游泳。

### 如厕，清洗手术部位

术后对阴道应进行特殊级别的清洗和适当涂抹抑菌药物。如厕时，应注意擦拭和清洗方式，如从前往后擦拭（这并不是术后护理的需求，而是女性生理结构的需求。）等。

大小便如厕后，仔细清洁会阴处。用淋浴喷头/马桶盖清洗，或用湿巾**从前到后**擦拭，避免将粪便冲入/擦入阴道。洗手后,用手来清洗生殖器,并用湿巾而不是卫生纸擦干。向阴蒂或阴唇(不要涂抹在阴道内)涂抹消毒抑菌药物。

在基本愈合后，如厕后不再需要即时冲洗。但需注意,女性小解后也需吸干(而非大力擦拭)以防止感染,不能像过去那样抖一抖就提裤子.如厕后擦拭时,均需遵守从前到后擦拭的要求。

在术者仍然频繁扩张的初期，应每天冲洗阴道。扩张频率降低后，冲洗可以减少到每周2-3次。阴道的异味或分泌物有些是冲洗可以解决的，有些则不是，请参考后文[阴道分泌物]栏目。

在大陆地区,除非自来水厂出现问题,一般情况下自来水往往经过氯气消毒杀菌,因此大多数情况可以放心使用自来水清洗术后位置。若有担心,可购置大包装的医用生理盐水。

判断是否出现如厕问题及处理：肿的状态即使正常地尿出来之后会被红肿皮瓣挡歪，没消肿等消肿之后才能判断。普通的尿路感染只是「女孩子的问题」，不是手术问题，普通泌尿专科可以解决。如果出现比较严重的左右前后歪斜，要拍图给手术医生询问严重与否。术后改证后如厕护理可以去普通医院做盆底肌训练（注：尤其是北医三院等可以由友跨医生引荐）也可阅读教程自己尝试

若您在手术中放置了留置导管或尿袋，该导管将在几天后拆下。在您对导尿管及其功能感到满意之前，检查引流管，确保其引流良好。如果你喝了足够的液体，袋子里应该有一些尿液。若有尿袋,始终将其放在下腹部水平以下——这是因为尿液通过重力排出——如果引流袋高于腹部水平，它将不会排出。护士会每天数次将收集袋倒进厕所。如果袋子里没有尿液收集，提醒护士检查导管或引流管是否有扭结。

如果您有以下任何问题，在住院期间请尽快告知护士和医生：

1. 尽管液体摄入良好，但没有尿流

2. 尿液中有大量血液或血凝块

3. 严重的腹部绞痛，

4. 有尿路感染的迹象（尿液混浊、发烧、疼痛）

拔除尿管后,排尿和憋尿需要经过适应和适应性训练.拔出尿管后,护士会要求您喝足够水并上厕所.放松身体,吸气后再呼气收腹有利于感受膀胱、避免盆底肌紧张,便于排尿.若腹腔疼痛不可忍受,尽快呼叫护士.

为减少尿路问题或尿路感染等的风险,尽量多喝水.

术后1-12月内尽量避免会阴部位电针脱毛。会辣到。

### 阴道分泌物与气味

在皮瓣术式下,阴道以皮瓣(包皮或阴囊皮)为衬里，皮肤菌群和一些阴道菌群在此繁殖；阴道症状的出现与任一特定菌种之间没有相关性。用来组成阴道壁的皮瓣有汗腺，表皮在新陈代谢中也会不断地脱落,亦有类似于"包皮垢"的物质在阴蒂周围、阴唇皱襞下和阴唇皱襞内积聚。若医生未将尿道大腺切除而是置于前庭大腺的位置,可能会有较为湿润的无色分泌物,在大多数情况下，分泌物很可能是皮脂，死皮或角蛋白碎片，以及存留的精液或润滑剂.这些物质及其混合物会发酵，并会导致浅黄色色斑。[^ucsf护理指南]

在乙状结肠等肠道术式下,阴道以乙状结肠等肠道为衬里,因此,可能有常见的肠分泌物(如肠液等)和肠异味等.

在腹膜术式下,无色分泌物可能较为多，需考虑如何避免其影响生活。

若有阴道分泌物含有有害微生物并进入尿道组织中,有可能会导致尿路感染。

当术后出现阴道异味及分泌物时,应当:

2. 首先要检查阴道内有没有内部感染，检查内部是否有脓疮病变或肉芽等，若有，及时就医
3. 确认是否有内部小伤口，如肉芽或阴道壁撕裂，这些伤口可能会偶尔出血。如果不定期冲洗,血液在微生物作用下可能产生异味与感染。可以通过观察扩张时模具上残留的分泌物的颜色来确认。
4. 皮瓣术式下,阴道气味可能随激素水平波动[^证据缺乏]
5. 若异味或分泌物持续且并非脓疮、病变、肉芽、出血造成的问题,用棉签擦拭掉粘稠的分泌物并用清水清洗会阴部.若菌群过度生长或失调,可考虑使用顺性别女性治疗阴道炎的药物清洗。

可购买卫生巾使用,以防止术后短期内出血和分泌物污染裤子. 尽量挑选大品牌、轻薄、材料好且无香料的卫生巾.

### 关注方面:性行为与敏感性

性行为应以保证舒适与安全为准则.

术后,性行为的快感及高潮一般可由对 保留的前列腺、阴道内埋置的龟头神经(仅部分医生声称)、阴蒂组织 的刺激达到。可在充分恢复和保证安全的情况下后尝试包括但不限于经阴道插入/纳入式性行为、摩擦阴蒂、经肛门性行为的方式进行性交或自慰，以达到性快感或高潮。术后的性快感有助于更好地适应外阴组织的变化，将其视为身体的一个组成部分，帮助达到自我接纳。

顺性别女性中发生率较高的性唤起障碍、性兴趣减退、性高潮障碍、性交痛(阴道痉挛)等亦会在术后跨性别女性中产生。在遇到上述情况时，进行性行为时应放松，与其他女性交流性经验，使用性玩具辅助等，以获得快感为目标。

在三个月内应尽量避免性方面想法,以免性唤起后会阴部位充血导致的疼痛、肿胀、愈合困难等.

应完全避免在六个月内进行经阴道插入/纳入式性行为。术后半年至一年，以阴道瘢痕软后进行经阴道插入/纳入式性生活为好；进行经阴道插入/纳入式性行为的一开始，完全插入/纳入可能是困难和痛苦的，插入行为如扩张等可能需要一段时间才能实现。最初进行经阴道插入/纳入式性生活时应使用润滑剂、使用性玩具或模具辅助、动作尽量温和，直至生理适应为止。扩张后尝试性交可能会有很大帮助(因扩张后阴道较为开放)。如果性交引起疼痛或困难，不要继续。

与顺性别女性相同，术后亦可通过摩擦、性玩具振动、抚摸阴蒂获得性快感及性高潮.进行阴蒂自慰或性交时,应保证接触阴蒂的身体部位或器具(性玩具)清洁,动作温和,性生活后清洗整个会阴,以避免阴部潮湿(术后跨性别女性亦可能有阴道分泌物)等造成的阴蒂、阴道、尿道感染等.

由于会阴部仍然是性感区,仍然可进行经肛门性行为.然而,由于睾丸切除、激素治疗等降低了体内雄性激素的浓度,前列腺不可避免地发生萎缩,且阴道壁一定程度地阻隔, 或将削减经肛门性行为的快感.[^无文献证明快感一定下降].

经肛门性行为有较高的HIV暴露风险，且阴道较顺性别女性弹性和分泌物较差，更容易在性行为中出血或撕裂。进行性行为时可使用较多的润滑剂减少出血风险，并采用PreP、安全套等方式做好个人防护，在发生高危性行为后及时获取阻断药物。有规律性生活的情况，应在公益机构或疾控中心定期检查。

外部:

- 确保你有足够的时间扩张
- 让环境尽可能舒适愉快。包括让床变得舒适，稍微撑高自己的上半身(不要撑得太高,否则将无法将扩张器插入所需的全部深度)，尽量让 气氛舒缓，把要使用的东西都放在手边。

- 在扩张器上套一个安全套，充分润滑扩张器

- 洗干净双手

## 术后并发症的预防和干预

如果在医院外感觉到不适或发生并发症，除非你的问题很明显需要紧急关注(如大出血等), 尽可能去联系为您SRS的医生或离您最近的可以进行SRS术后修复的SRS医生。若您咨询的医生没有性别肯定医疗的经验，可能会在心理和生理上造成伤害。得到SRS医生医生的转荐后,您才应当找一般的非性别肯定医疗医生如妇科医生等，解决您的问题。

术后严重并发症一般在医院住院监视期间就得到了妥善的解决(因此,本文建议选择有处理并发症水平的医院)。若遇到罕见严重并发症(如手脚麻痹、小腹切口感染,结肠接口感染,肠粘连肠梗阻),请马上联系SRS医生。若为您进行手术的医生拒绝为此负责,请联系其余可以进行SRS修复的医生并咨询医疗保险赔付事宜等。

尿道阴道瘘以尿液从阴道泄漏的形式出现。大多数病例不需要或不需即刻干预，并且多数情况下患者仍然可以自制。

阴道中的肉芽组织是延迟愈合的结果，并较为常见。术后早期阶段频繁扩张的需求会通过对肉芽区域造成反复创伤而加剧此问题。多数情况下，这会由于频繁扩张的需求随时间下降而逐步愈合。如症状持续，遵循医生建议考虑定期烧灼治疗和局部乳膏治疗.长期来看，阴茎皮肤内衬的阴道应极为耐用。

阴道出血在术后较为常见，更常见的是尿道周围出血。即使在愈合期过后,也可能出现。应尽量避免便秘(通过摄入可溶性膳食纤维等)以避免排便时过于用力。出血的检查方法:保持冷静(以避免血压升高)和静止，试着找出源头和判断出血量。用清水或生理盐水清洁会阴部,判断出血部位究竟是阴蒂、尿道、阴道内外还是肛门处(可能缘由痔疮或直肠出血等其他与SRS无关的情况)。并清洁出血处附近.若并非在内部出血,尝试用酒精棉球按压。若仍然无法止血, (若非肛门出血)可联系为您SRS的医生或离您最近的可以进行SRS术后修复的SRS医生。（若在乙状结肠术式下,经由肛门出血亦可能与SRS有关）在模具上或内裤上出现的血液,仅在鲜红色(明显刚刚出现)或血液量大量或频繁出血的情况需要注意。

休止期脱发：是一种暂时性脱发，通常发生在压力、休克或创伤事件后。它通常发生在头皮顶部。休止期脱发与称为斑秃的脱发症不同,也与脂溢性脱发(雄激素性秃法)区别。一个人的大量头发可能会脱落，但通常是暂时的，头发通常会重新长出来。性别重置手术(SRS)是侵入性过程,过程中的压力可能导致脱发,保持放松的心情有助于改善。尤其是不要一掉头发就害怕自己反雄雄秃。

VTE(深静脉血栓)：在目前,几乎所有诊所跨性别女性必须在外科手术前后停止雌激素治疗，特别是本身有高VTE风险的术者. 可能因历史上使用炔雌醇(EE)的HRT(激素替代治疗)带来较高的血栓风险,或因部分雌二醇给药途径带来略低的给药风险. 术中措施及术后长期卧床,都会放大这些风险. 如对此有异议或自身VTE风险较高,应与外科医生进行充分的知情同意讨论,并及早咨询医生开具给药途径VTE风险更小的激素补充药物。

“反雄”：术(性腺切除或SRS)后的女性一般上不用担心睾酮问题，但是在非常罕见的情况下，仍然有较低的风险经由肾上腺分泌等途径导致睾酮(T)或二氢睾酮(DHT)升高。若出现反雄，检查游离睾酮和二氢睾酮水平，并咨询有资格指导激素治疗的医生开具药物帮助缓解。

术(性腺切除或SRS)后，为避免围绝经期症状 如潮热、骨质疏松、心脑血管疾病 风险等, 仍然需要补充雌激素。咨询有资格指导激素治疗的医生开具调整药物。

双促（LH/FSH）升高：这是常见的，是垂体找不到蛋而发生惊慌失措的行为。然而，术后双促或有概率可标定雌激素水平，可咨询有资格指导激素治疗的医生。

在皮瓣术式下,由于阴道是皮肤衬里的，因此存在发生与阴茎和阴囊皮肤（鳞状细胞，基底细胞， 黑色素瘤）上相同皮肤癌的风险。尤其是感染了人乳头瘤病毒HPV如尖锐湿疣等(北美调查中.跨性别女性是高危性行为和HPV感染的高发人群)的情况下,这些癌症的风险会骤升。应注意检查以排查该风险。其他皮肤病也会影响皮瓣阴道，应进行类似治疗。

### Q:我不进行PIV(插入式行为),需要扩张吗?

近些年来,越来越多跨性别女性敢于接受自己的女性同性恋/泛型恋的多重性少数身份并表现出来。

1. 一位医生认为仍然需要扩张到一个可以进行经阴道清洗的程度，以保证干净卫生和感染风险小。在无法判断是否已经扩张到了这一程度时，仍然需要进行一定程度的扩张.
2. 部分医生认为即使是女同/泛性恋的跨性别女性，也会在阴道萎缩时经历心理波动和不适。或因为可能有未预料的肉芽、增生风险而建议进行扩张

### Q:我可以做经阴道检查吗?

在跨性别女性中创建的新阴道的解剖结构与先天性阴道的不同之处在于它具有盲端，缺少子宫颈或 周围的穹窿，并且可能方向更加靠后。
